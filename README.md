# 📚 Spring Boot - Estudos e Exemplos

Este repositório foi criado com o objetivo de documentar minha evolução nos estudos de **Java** e **Spring Boot**, reunindo exemplos práticos, exercícios e pequenos projetos desenvolvidos durante o aprendizado.

A ideia é utilizar este espaço como um laboratório para testar conceitos, praticar boas práticas de desenvolvimento e consolidar os conhecimentos adquiridos.

---

## 🎯 Objetivos

- Praticar o desenvolvimento de APIs REST com Spring Boot.
- Aprender a utilizar os principais módulos do ecossistema Spring.
- Desenvolver CRUDs utilizando e, posteriormente, sem banco de dados.
- Estudar arquitetura em camadas (Controller, Service e Repository).
- Aplicar conceitos de Programação Orientada a Objetos (POO).
- Evoluir gradativamente para projetos mais completos.

---

## 📂 Conteúdo

Este repositório poderá conter exemplos de:
